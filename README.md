# PyTorch Exercises

A collection of hands-on PyTorch exercises to build foundational skills with tensors and core operations.

## Exercises

| # | Topic | Description |
|---|-------|-------------|
| 1 | Tensor Fundamentals | Creating tensors from lists and NumPy arrays, basic tensor operations |
| 2 | Tensor Operations | Reshaping, viewing, and slicing tensors |
| 3 | Tensor Math | Element-wise operations (addition, subtraction, multiplication, division, modulo, exponents) |
| 4 | Neural Networks | Building a basic feedforward neural network with multiple layers using nn.Module |

## Setup

```bash
python -m venv pytorch_venv
pytorch_venv\Scripts\activate   # Windows
pip install -r requirements.txt
```

## Requirements

- Python 3.8+
- PyTorch >= 2.0.0
- NumPy
