# PyTorch Exercises

A collection of hands-on PyTorch exercises to build foundational skills with tensors and core operations. Exercises are generated by Claude Opus 4.6.

## Exercises

| # | Topic | Description |
|---|-------|-------------|
| 1 | Tensor Basics | Creating tensors, element-wise operations, inspecting shape/dtype/device |
| 2 | Reshaping & Indexing | `torch.arange`, reshaping, row/element extraction, slice notation |

## Setup

```bash
python -m venv pytorch_venv
pytorch_venv\Scripts\activate   # Windows
pip install -r requirements.txt
```

## Requirements

- Python 3.8+
- PyTorch >= 2.0.0
- NumPy
